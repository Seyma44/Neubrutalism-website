## Getting Started
```bash
git clone https://github.com/Seyma44/Neubrutalism-website/
cd Neubrutalism-website
```
Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
